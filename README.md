# Page Cleaver
Splits the web page into two panes that can be individually scrolled.

The release CRX may not work. If it doesn't work, just clone the repo and add the extension in developer mode as an unpacked extension.

This is just a quick extension I thought would be funny to make. My brother gave me the original idea, and it wasn't too hard. There are no guarantees this is going to work on any browser other than Chrome and any day after the day I wrote it.
